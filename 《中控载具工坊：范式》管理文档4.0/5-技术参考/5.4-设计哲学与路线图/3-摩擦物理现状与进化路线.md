# 3. 摩擦物理现状与进化路线

## 当前模型（街机模式 · Binary Grip · 2026-06-17）

```
每轮独立判定：
  driveImpulse = (torqueB/n / radius) × dt
  maxGrip      = μ × max(springImpulse, nm × dt × MIN_IMPULSE_MULTIPLIER)
  
  driveImpulse ≤ maxGrip → 全量传递（抓地）
  driveImpulse > maxGrip → 截断到 maxGrip（打滑）
```

**二值库伦摩擦**，无摩擦圆预算共享，无载荷转移，无侧偏模型。
不再有总需求/预算比率缩减，无纵向/横向力妥协。

覆盖层显示：`ratio = driveImpulse / maxGrip - 1.0`，全轮平均 × 100%。

## 旧模型（已禁用）

### 摩擦系数随负载递减

真实轮胎的摩擦关系近似：$F_{grip} \approx \mu_0 \times N^{0.85}$

| 场景 | 库伦模型 ($N^{1.0}$) | 真实轮胎 ($N^{0.85}$) |
|------|-------------------|--------------------|
| 50 MXT 轻车 | 抓地 = 50μ | 抓地 ≈ 28μ |
| 500 MXT 重车 | 抓地 = 500μ × 10 倍！| 抓地 ≈ 224μ × 8 倍 |
| 重量 10× | 抓地力 10× | 抓地力 ≈ **8×** |

## 未来轮胎模型设计（概念）

```
轮胎数据（Tire 物品组件）
├─ 宽度 w          → 接触面宽度
├─ 扁平比 profile   → 侧壁高度 = w × profile
├─ 内压 pressure    → 抵抗变形的刚度
└─ 橡胶刚度 k       → 单位变形的回弹力

物理计算（每 tick 每轮）：
├─ 1. 负重 N = 弹簧力（已有）
├─ 2. 接触面长 L = √(N / (pressure × w))
├─ 3. 接触面积 A = w × L
├─ 4. 摩擦系数 μ_eff = μ_0 × (A / A_ref)^(-0.15)
├─ 5. 抓地力 F_grip = μ_eff × N（替换 frictionBudget）
├─ 6. 滚动阻力 F_rr = C_rr × N + k_hys × A × v
└─ 7. MIN_IMPULSE_MULTIPLIER 进一步降至 ~10
```

## 轮胎气压可调性

```
低压（1.5 bar）→ 接触面大 → 抓地力↑ → 滚动阻力↑ → 越野/沙地
标准（2.2 bar）→ 均衡            → 公路
高压（3.0 bar）→ 接触面小 → 抓地力↓ → 滚动阻力↓ → 高速巡航
```
