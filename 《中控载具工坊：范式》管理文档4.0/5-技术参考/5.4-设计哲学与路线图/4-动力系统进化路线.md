# 4. 动力系统进化路线

## 当前架构（街机模式 · 2026-06-17 确立）

**完整取代了之前所有离合器/摩擦圆/载荷转移等专业级模型。**

```
油门 → EngineModel(扭矩a, RPMa) → TransmissionModel(扭矩b, RPMb)
  → Binary Grip(每轮 min(扭矩/n, μ×N))
  → 走或不走
```

核心设计：
- **发动机完全独立**：油门直控 RPM，不受变速箱/轮速反向影响
- **变速箱纯数学变换**：无离合器状态机，扭矩b = 扭矩a × 齿比
- **Binary Grip 二值抓地**：每轮独立，无预算共享

换挡真空期 6 tick + 降档自动补油（rev-match）保留。

## 已弃用的旧模型

旧问题（引擎-车轮脱耦、换挡 RPM 同步等）已被街机模式架构从根本上解决：
- 引擎不再需要与车轮耦合——RPM 由油门直控
- 换挡时 RPM 由油门决定，不受齿比变化影响
- 变速箱仅做数学乘除，无物理耦合行为
