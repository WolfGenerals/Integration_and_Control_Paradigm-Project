# 0. 命名由来与星球概况

## 命名由来

```
MC → 爱慕C → C→X（舌头姿态不同）→ 安慕希 → 安慕星 (MX)
```

**安慕**：安宁、慕恋——一颗物质蓬松、物理自洽的和平星球。  
**MX**：Minecraft × Sable 的交叉宇宙，也是「梦想」(Meng Xiang) 的缩写。

## 安慕星基本设定

| 属性 | 值 |
|------|-----|
| 名称 | 安慕星 (Ānmù Xīng, MX) |
| 别名 | MX-1, 「方块世界」 |
| 空间维度 | 三维，最小单位 1×1×1 MX-meter (MXm) |
| 物质形态 | 方块 (Block) 为基本单元，不可再分 |
| 物理引擎 | 底层基于 Rapier（Rust），上层经 Sable 桥接 |
| 时间流速 | 20 tick/s（50 ms/tick），Rapier 子步进未知 |

## 与地球的对应关系

| 维度 | 地球 (🌍) | 安慕星 (⭐) |
|------|----------|-----------|
| 长度 | 1 m | 1 MXm ≈ 1 m（视觉一致） |
| 体积 | 1 m³ | 1 方块 = 1 MXm³ |
| 质量 | 1 kg | 1 MX-ton (MXT) ≈ 1400 kg |
| 密度基值 | 水 = 1000 kg/m³ | 泥土 = 1.0 MXT/m³ |
| 时间 | 1 s | 1 MXs ≈ 1 s（但离散为 20 tick） |
| 重力 | 9.81 m/s² | 推测 ≈ 20 MXm/MXs²（玩家跳跃手感反推） |

> **核心法则：安慕星的物理不需要与地球一一对应，它内部自洽即可。**
