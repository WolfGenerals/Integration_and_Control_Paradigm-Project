# 十七、动态载荷转移

### 核心公式

```java
// 每 tick：保存 prevLocalVelocity → 计算 accelZ/accelX
// 通过驾驶舱位置确定轮位 normZ/normX（归一化到 [-1, 1]）

double longTransfer = -accelZ * COG_HEIGHT / (g * HALF_WHEELBASE) * normZ;
double latTransfer  = -accelX * COG_HEIGHT / (g * HALF_TRACK) * normX;
loadTransfer = (longTransfer + latTransfer) * LOAD_TRANSFER_SENSITIVITY;

// 调整摩擦基数
double adjustedSpringImpulse = springImpulse * (1.0 + loadTransfer);
```

### 物理对应
| 场景 | accelZ | 前轮 | 后轮 |
|------|:------:|:----:|:----:|
| 加速前进 | + | 减载 | 增载 |
| 刹车 | - | 增载 | 减载 |

### 侧向符号（已修复 ✅）

**修复于 06-08**：侧向载荷转移符号已从 `-accelX` 修正为 `+accelX`。

当前正确公式：
```java
double latTransfer = +accelX * COG_HEIGHT / (g * HALF_TRACK) * normX;
```

#### 物理直觉速查
| 场景 | accelX | 左轮 (normX < 0) | 右轮 (normX > 0) |
|:----:|:------:|:----------------:|:----------------:|
| 左转 | > 0 | 减载（抬起） | 增载（压下） |
| 右转 | < 0 | 增载（压下） | 减载（抬起） |

### 参数
| 常量 | 值 | 说明 |
|------|-----|------|
| `LOAD_TRANSFER_SENSITIVITY` | 0.3 | 每 g 转移载荷比例 |
| `COG_HEIGHT` | 0.8 | 重心高度（格） |
| `HALF_WHEELBASE` | 1.5 | 半轴距 |
| `HALF_TRACK` | 1.0 | 半轮距 |
| 钳制范围 | ±0.8 | 防止单轮完全离地 |
