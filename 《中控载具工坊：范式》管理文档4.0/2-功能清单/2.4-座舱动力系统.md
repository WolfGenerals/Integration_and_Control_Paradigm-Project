# 座舱动力系统

## 架构概览

```
Player Input → CockpitBE.tick()
  → EngineModel: 质量自适应扭矩 + 扭矩曲线 + 连续油门 + 负载模型
  → TransmissionModel: 档位/换挡/差速分配
  → SuspensionTestBE: 各轮施加扭矩
```

所有计算模型为纯静态函数，状态由 `CockpitBlockEntity` 管理。

## 发动机模型 (Engine)

| 参数 | 值 |
|------|:---:|
| 怠速 RPM | 800 |
| 红线 RPM | 6,000 |
| 基础扭矩 ENGINE_TORQUE | 30.0 Nm |
| 质量自适应比率 TORQUE_WEIGHT_RATIO | 0.02 |

**质量自适应扭矩**：`effectiveTorque = totalMass × g × 0.02`，下限 `ENGINE_TORQUE × 0.5 = 15 Nm`。重车扭矩大、轻车扭矩小，推重比一致。

**扭矩曲线**：使用 sin() 曲线修正 RPM 依赖扭矩，中段 ~3,400 RPM 峰值。

**连续油门**：throttleLevel 0.0~1.0 连续调节，三段式衰减（加速/滑行/主动减油）。

**负载模型**：`loadFactor = 力需求 / 最大力`，>1.0 时拖低转速。

## 变速箱 (Transmission)

| 档位 | 齿比 | 最终传动比 (×14.0) |
|:----:|:----:|:------------------:|
| 1 档 | 4.0 | 56.0 |
| 2 档 | 2.5 | 35.0 |
| 3 档 | 1.6 | 22.4 |
| 4 档 | 1.2 | 16.8 |
| 5 档 | 1.0 (直接档) | 14.0 |
| R 档 | -3.5 | -49.0 |
| N 档 | — | 空档 |

**换挡**：Q 升档 E 降档，上升沿检测 + `GearShiftC2SPacket`，换挡时同步 RPM。

**差速分配**：同转速均摊扭矩到所有轮子。

**降级兼容**：无驾驶舱 SubLevel 回退固定 RPM/Torque (400/80)。
