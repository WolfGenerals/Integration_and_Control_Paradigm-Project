# 座舱动力系统

## 架构概览（街机模式 · 2026-06-17 确立）

```
Player Input (油门)
  ↓
EngineModel.computeThrottleControlledRun()  ← 发动机完全独立
  RPMa = IDLE(800) + throttle × (MAX - IDLE)
  扭矩a = ENGINE_TORQUE × torqueCurve(RPMa)  ← 纯 RPM 函数
  ↓
空档: torquePerWheel = 0（变速箱断开）
在档: TransmissionModel.computeOutput()  ← 纯数学变换
  扭矩b = 扭矩a × 齿比 × FDR
  RPMb  = RPMa / 齿比 / FDR × sign
  换挡真空期 6 tick → 扭矩b = 0
  ↓
SuspensionTestBE: Binary Grip 判定
  驱动冲量 ≤ μ × 法向冲量 → 全量传递（抓地）
  驱动冲量 > μ × 法向冲量 → 截断（打滑）
```

所有计算模型为纯静态函数，状态由 `CockpitBlockEntity` 管理。
**发动机与变速箱完全解耦**，挂档不拉低转速，轮速不反向约束发动机。

## 发动机模型 (Engine)

| 参数 | 值 |
|------|:---:|
| 怠速 RPM | 800 |
| 红线 RPM | 6,000 |
| 基础扭矩 ENGINE_TORQUE | 200.0 Nm |
| 油门控制 | 连续双向调节（Home加油/End减油） |

**油门直控**：RPM 由 throttle 直接定位，非扭矩积分演化。
- throttle=0% → 800 RPM | throttle=50% → 3,400 RPM | throttle=100% → 6,000 RPM

**扭矩曲线**：使用 sin() 曲线修正 RPM 依赖扭矩，中段 ~3,400 RPM 峰值（200 Nm）。
怠速和红线均为 160 Nm（×0.80）。

**油门 100% 已包含内部损耗系数**，无额外摩擦项。覆盖层内摩擦/净扭矩不再显示。

## 变速箱 (Transmission)

| 档位 | 齿比 | 最终传动比 (×14.0) |
|:----:|:----:|:------------------:|
| 1 档 | 4.0 | 56.0 |
| 2 档 | 2.5 | 35.0 |
| 3 档 | 1.6 | 22.4 |
| 4 档 | 1.2 | 16.8 |
| 5 档 | 1.0 (直接档) | 14.0 |
| R 档 | -3.5 | -49.0 |
| N 档 | — | 空档 |

**纯比率变换**：无离合器、无状态机、无滑差。齿比仅做数学乘除。

**换挡**：Q 升档 E 降档，6 tick 换挡真空期（动力中断），降档自动补油（rev-match）。

**差速分配**：所有轮子均分扭矩b。（差速器偏置已禁用）

## Binary Grip 抓地力

取代旧摩擦圆模型。每轮独立判定：

```
每 tick:
  驱动冲量 = 扭矩b/n / 半径 × dt
  抓地极限 = μ_轮胎 × μ_地面 × 法向冲量
  
  驱动冲量 ≤ 抓地极限 → ✅ 全量传递（抓地）
  驱动冲量 > 抓地极限 → ❌ 只传极限值（打滑）
```

无摩擦圆预算共享、无载荷转移、无纵向/横向力妥协。

## 暂禁用的子系统

| 系统 | 原因 |
|------|------|
| 摩擦圆模型 | Binary Grip 取代 |
| 动态载荷转移 | 所有轮子用弹簧静载 |
| 空气阻力（二次方阻尼） | 扭矩曲线 × 齿比已天然限速 |
| 差速器偏置 | 所有轮子均分扭矩 |
| Brush 侧偏模型 | 侧偏力暂归零 |
| 横移轮独立路径 | 等待智能映射系统 |
| 降级回退 P 控制器 | 无驾驶舱不驱动 |
