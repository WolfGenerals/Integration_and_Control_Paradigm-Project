# 3C 控制系统 — 载具 (Cockpit)、摄像机 (Camera)、控制 (Control)

## 驾驶舱 (Cockpit)

- **下格** (`CockpitBlock`)：炼药锅形状（底部实心 + 四壁），实现 IBE 接口提供 ticker
- **上格** (`CockpitUpperBlock`)：脚手架形状
- 放置时自动向上延伸两格，破坏任一部分连带破坏另一半
- 驾驶舱组定义使用 `Set<Block>` 直接比较，不依赖 Tag 系统

## 上车/下车系统 (MountSystem)

- F 键射线检测 → `Sable.HELPER.getContaining()` 查找 SubLevel → 检查独占/唯一性 → 隐藏玩家
- **宽泛射线命中**：无需对准驾驶舱，命中 SubLevel 任意方块即可
- **下车全场景覆盖**：F 键/断线/死亡/传送均安全下车，UUID 查找不依赖玩家位置
- 下车坐标系问题（玩家位置 vs SubLevel 底层坐标）已于 06-13 修复

## 玩家隐藏系统

- `RenderPlayerEvent.Pre.setCanceled(true)` + `setInvisible(true)` — 完全不可见
- `noPhysics=true` + 服务端零碰撞箱 — 免疫伤害和碰撞
- 断线重连 NBT 清理：`EntityJoinLevelEvent` 清理残留

## 轨道摄像机 (TrackCamera)

- `CameraMixin` 在 `Camera.setup()` @Tail 注入
- 球面环绕 SubLevel 焦点，`horizontalDist < 1e-4` 时沿用 `entity.getYRot()` 防奇点
- 自适应高度/距离由 `Config.java` 控制（两个独立开关 + 偏移量）
- **自适应高度抖动修复（06-15）**：自适应高度模式下，焦点 Y 使用 `renderPos.y()`（插值平滑）替代 `bbox.maxY()`（物理 AABB 非插值），消除载具垂直运动时的摄像机抖动

## WASD 智能映射

- FACING 投票 → 宽度轴 → 质心 Z → 每轮智能按键分配
- **Car Mode**：南北半区转向镜像
- **Reverse**：W↔S, A↔D 交换 + 引擎层齿比符号反转
- **Toggle**：开启→应用 Car Mode，关闭→清除 smartKey
- **LocalSwapSmartKeys**：发送 REVERSE 后立即在客户端交换智能键，消除网络延迟

## 哨兵摄像机模式 (StationaryCam)

V 键切换：摄像机冻结在世界坐标，持续锁定载具中心；期间停止瞄准坐标发送但保留驾驶/开火能力；退出时自动面向载具速度方向。
