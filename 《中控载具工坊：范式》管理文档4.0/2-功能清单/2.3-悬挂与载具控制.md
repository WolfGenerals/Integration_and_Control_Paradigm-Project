# 悬挂与载具控制

## 悬挂测试方块 (SuspensionTestBlock)

独立于 Create 动力系统的悬挂+轮子物理测试方块。编译时常量控制全部视觉偏移，BE 承载：

- 悬挂物理（弹簧-阻尼器模型，刚度 400 N/m per nm，阻尼 40）
- 摩擦圆物理模型
- 手刹物理模型
- 转向控制
- 按键配置系统
- 动态载荷转移
- Brush 轮胎侧偏模型
- 轮胎物理参数系统
- 二次方速度阻尼
- 发动机-轮速耦合数据源

## 摩擦圆物理模型

纵/侧向力共享摩擦预算，超出等比例缩减。核心公式：

```
frictionBudget = μ × max(springImpulse, nm × dt × MIN_IMPULSE_MULTIPLIER)
totalDemand = sqrt(longitudinal² + lateral²)
ratio = totalDemand / frictionBudget
if ratio > 1: 纵/侧向力等比例缩减
```

关键参数：`MIN_IMPULSE_MULTIPLIER = 30`（06-08 从 500 降下来）

## 手刹物理模型 (v3 最终方案)

轮子抱死 + 纯滑动摩擦，使用 `springImpulse` 真实法向力基数，减速度 ≈ 0.35g 不钉地。

## 动态载荷转移

加速/刹车/转弯时重心偏移改变各轮法向载荷。核心参数：`LOAD_TRANSFER_SENSITIVITY`、`COG_HEIGHT`。

## Brush 轮胎侧偏模型

侧偏角峰值约 4.5°，过峰值后抓地崩溃模拟漂移。侧偏角钳制 ±45° + `totalSpeed > 1.0` 阈值防抖。

## 轮胎物理参数系统

06-09 精简为编译时常量，玩家唯一可调 = 胎压。胎压影响滚动阻力/爆胎风险/抓地力。未来通过 TireType 注册表扩展款式区分。
