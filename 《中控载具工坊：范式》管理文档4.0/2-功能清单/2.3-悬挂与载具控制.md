# 悬挂与载具控制

## 悬挂测试方块 (SuspensionTestBlock)

独立于 Create 动力系统的悬挂+轮子物理测试方块。编译时常量控制全部视觉偏移，BE 承载：

- 悬挂物理（弹簧-阻尼器模型，刚度 400 N/m per nm，阻尼 10）
- Binary Grip 抓地力判定
- 手刹物理模型
- 转向控制
- 按键配置系统
- 轮胎物理参数系统
- 发动机-轮速耦合数据源

## Binary Grip 抓地力（2026-06-17 取代摩擦圆）

二值抓地力模型。每轮独立判定，无预算共享：

```
driveImpulse = (torqueB/n / radius) × dt
maxGrip      = μ × max(springImpulse, nm × dt × MIN_IMPULSE_MULTIPLIER)

driveImpulse ≤ maxGrip → 全量传递
driveImpulse > maxGrip → 截断到 maxGrip（打滑）
```

摩擦需求比（覆盖层显示）：
```
ratio = driveImpulse / maxGrip - 1.0
平均 × 100%
-100% = 无负载, 0% = 饱和, +100% = 2×打滑
```

## 手刹物理模型 (v3 最终方案)

轮子抱死 + 纯滑动摩擦，使用 `springImpulse` 真实法向力基数，减速度 ≈ 0.35g 不钉地。

## 轮胎物理参数系统

06-09 精简为编译时常量，玩家唯一可调 = 胎压。胎压影响滚动阻力/爆胎风险/抓地力。未来通过 TireType 注册表扩展款式区分。

## 暂禁用的子系统

| 系统 | 原因 |
|------|------|
| 摩擦圆物理模型 | Binary Grip 取代 |
| 动态载荷转移 | 所有轮子用弹簧静载 |
| Brush 轮胎侧偏模型 | 侧偏力暂归零 |
| 二次方速度阻尼（空气阻力） | 扭矩曲线限速已足够 |
| 差速器偏置 | 所有轮子均分扭矩 |
| 横移轮独立路径 | 等待智能映射系统 |
