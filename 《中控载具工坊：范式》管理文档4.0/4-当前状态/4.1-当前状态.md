# 当前状态

> 更新于 2026-06-17（v6·街机模式动力系统重构）。仅维护活跃问题/待办/已验证行为。已完成功能见 `2-功能清单/2.1-功能总表.md`。

## 街机模式动力系统（2026-06-17 里程碑）

**正式替代所有旧专业级模型。** 新的核心三元组：

1. **油门直控发动机** — RPM = IDLE + throttle × (MAX - IDLE)，扭矩纯 RPM 函数
2. **纯比率变速箱** — 无离合器状态机，扭矩b = 扭矩a × 齿比
3. **Binary Grip 抓地力** — 每轮独立 min(驱动力, μ×N)，无预算共享

**已移除/禁用：** 摩擦圆、离合器锁止状态机（CLUTCH_*常量全部删除）、载荷转移、空气阻力、Brush 侧偏模型、差速器偏置、横移轮路径、降级回退P控制器。

覆盖层摩擦需求改为 Binary Grip 比率 `(驱动/极限 - 1.0) × 100%`，全轮平均显示。

| onWorldLoad() 从未被调用 | 创建 WorldLoadHandler 并注册到 NeoForge.EVENT_BUS |
| resolvePolicy 在 rayTraceSubLevels 中调用两次 | 缓存 policy 变量避免重复调用 |
| CAMERA_AIM 不必要的检查 | 提前返回跳过 sameVehicle/sameFaction 检查 |

## ✅ 已验证的代码行为（关键部分）

- `setPos()` 替代 `teleportTo()` 不触发位置包 — 无 lerpSteps 拉回
- `SUBLEVEL_OCCUPANTS` 断线/销毁/下车均正确清理
- 降级兼容：无驾驶舱回退固定 RPM/Torque
- 5 档直接档轮端 RPM = 发动机 RPM
- 位置模式 PD 伺服每 tick position-mode setMotor 正常响应
- Create RPM 阶跃特性：0→±16→±88→0 均在 1 tick 内完成
- setMotor velocity mode 在 RotaryConstraint 上无效（必须 position mode）
- AffiliationRegistry 注册/注销/组管理/玩家-载具绑定 全部可用
- ComponentRegistry 三层索引 / ComponentHost 自动注册注销 全部可用
- 瞄准系统全链路延迟优化：ComponentRegistry O(1) + driveImmediate 消除空等
- **部件损坏简化版**：`PartDamageCache.damageBlock(Level, Vec3, float)` 单入口，使用 `findSubLevelAt()` 直接查找命中点 SubLevel；不做归属排除，所有 SubLevel 一视同仁
- **服务端实体伤害射线检测**：`WeaponFireC2SPacket.handle()` 用 `ProjectileUtil.getEntityHitResult()` 沿 `origin→hitPos` 做射线-实体交点检测（替代 AABB 包围盒），CRIT 粒子在精确交点 `preciseHit` 而非实体中心
- **弹道渲染**：BulletTrailRenderer 在 `AFTER_LEVEL` 阶段用 `RenderType.LINES` + 摄像机相对坐标 + `endLastBatch()` 渲染（与 AxisLineRenderer 模式一致）；炮口位置每帧从 `renderPose(partialTick)` 动态计算消除抖动
- **渲染/服务端原点分离**：客户端 `fireSingleTurret()` 计算两个原点——`renderOrigin`（无偏移，用于弹道渲染，保持平滑）和 `origin`（含砂轮速度偏移 × timeScale=0.5，用于服务端伤害判定，提高移动中命中准确性）
- **性能优化 06-15 终版**：
  - 热路径 `LOGGER.info()` 降级为 `debug`（`PartDamageCache`/`WeaponFireC2SPacket`/`SableBlockHelper`）
  - 客户端 `performRaycast` 从每 tick 改为每 2 tick（`raycastCooldown = 2`）
  - `raycastGeneric` SubLevel 循环加入 AABB 快速剔除
  - 移除速度偏移补偿代码（50ms/100ms 预测均取消）
  - AffiliationRegistry 新增 `VEHICLE_TO_DIRECT_MEMBERS` 索引、`WRITE_LOCK` 同步锁、`CAMERA_AIM` 快速路径
- **自适应摄像机高度抖动修复**：`CameraMixin` 自适应高度模式焦点 Y 使用 `renderPos.y()`（插值平滑）替代 `bbox.maxY()`（物理 AABB 非插值），消除载具垂直运动时的抖动
- **WorldLoadHandler**：监听 `LevelEvent.Load`，清空 AffiliationRegistry + ComponentRegistry，等待 NBT 重建
- **Veil 依赖**：`compileOnly "foundry.veil:veil-neoforge-1.21.1:4.1.4"`，仓库 BlameJared Maven
