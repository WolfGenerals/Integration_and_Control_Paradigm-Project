### 仓库配置

```groovy
// build.gradle
repositories {
    maven { url "https://maven.createmod.net" }          // Create + Flywheel
    maven { url "https://raw.githubusercontent.com/Fuzss/.../maven" }  // ForgeConfigAPIPort
    exclusiveContent {
        forRepository { maven { url "https://api.modrinth.com/maven" } }
        filter { includeGroup "maven.modrinth" }
    }
    flatDir { dir 'libs/extracted' }                     // 本地 Sable/Simulated JAR
}
```

## Simulated 项目调用点速查

| 需求 | 引用路径 |
|------|---------|
| 查找 SubLevel | `Sable.HELPER.getContaining(level, pos)` |
| 获取 SubLevel 位姿 | `subLevel.logicalPose()` → `Pose3dc` |
| 获取所有已加载 chunk | `subLevel.getPlot().getLoadedChunks()` |
| 获取中心方块 | `subLevel.getPlot().getCenterBlock()` |
| 施加力/扭矩 | `RigidBodyHandle.of(subLevel)` + `ForceTotal` |
| 获取速度 | `Sable.HELPER.getVelocity(level, position)` |
| 质量数据 | `subLevel.getMassTracker()` → `MassData.getMass()` |
| 获取 Tag 常量 | `simulated/.../index/SimTags.java` |
| 所有方块注册名 | `simulated/.../index/SimBlocks.java` |
| 所有物品注册名 | `simulated/.../index/SimItems.java` |
| 物理回调 | `SableEventPlatform.INSTANCE.onPhysicsTick()` |
| 装配/拆卸 | `SimAssemblyHelper.assemble()` / `disassembleSubLevel()` |

## NeoForge 1.21.1 迁移注意事项

### KeyMapping 构造器签名
```java
// 1.21.1 — 键码需要用 InputConstants.Type 包装
new KeyMapping("key.name", KeyConflictContext.IN_GAME,
    InputConstants.Type.KEYSYM, GLFW.GLFW_KEY_F, "category")
```

### CustomPacketPayload 网络系统
1.21.1 使用新的 payload 网络系统，替代旧的 `SimpleChannel`。详见 `2-功能清单/2.5-调试与监控.md`。

### EventBusSubscriber 事件总线选择
| 事件类型 | 总线 | 写法 |
|---------|------|------|
| RegisterPayloadHandlersEvent | MOD | `@EventBusSubscriber(modid = MODID)`（默认） |
| ServerTickEvent.Post | GAME | `@EventBusSubscriber(modid = MODID, bus = Bus.GAME)` |
| ClientTickEvent.Post | GAME | 用 `value = Dist.CLIENT` 区分 |
