# 代码文件索引

> 每文件一句话职责。按功能模块分组。

## 入口与配置

| 文件 | 职责 |
|------|------|
| `iac_p.java` | `@Mod("iac_p")` 主类。注册方块/物品/实体/创造标签页/事件 |
| `iac_pClient.java` | 客户端入口。注册配置屏幕、按键映射、客户端事件 |
| `Config.java` | 配置类：自适应高度/距离开关、摄像机高度偏移/距离/Y 轴反转 |
