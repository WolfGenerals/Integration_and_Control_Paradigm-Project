# 3. 变速箱与换挡系统

### 齿比

| 档位 | 齿比 | 最终传动比 (×14.0) | 6000 RPM 目标速度 | 用途 |
|:----:|:----:|:-----------------:|:-----------------:|------|
| R | -3.5 | -49.0 | — | 倒车 |
| N | — | — | 0 | 空档 |
| 1st | **4.0** | **56.0** | **6.3 m/s = 23 km/h** | 起步/爬坡 |
| 2nd | **2.5** | **35.0** | **10.1 m/s = 36 km/h** | 低速 |
| 3rd | **1.6** | **22.4** | **15.8 m/s = 57 km/h** | 中速 |
| 4th | **1.2** | **16.8** | **21.0 m/s = 76 km/h** | 高速 |
| 5th | **1.0** | **14.0** | **25.2 m/s = 91 km/h** | 巡航（P 误差抬到 ~120） |

> **主减速比** `FINAL_DRIVE_RATIO=14.0`（06-08 新增）：所有档位齿比乘以该值得到最终传动比。
> 5 档目标 91 km/h，P 控制器稳态误差 + 拖拽阻力将极速收敛到 ~120 km/h。

### 变速箱公式

```java
wheelRpm = (engineRpm / |ratio|) × sign(ratio)
totalWheelTorque = effectiveTorque × |ratio|   // ← 改用 effectiveTorque（质量自适应）
perWheelTorque = totalWheelTorque / numWheels
```

### 换挡序列

```
Q 升档: R → N → 1 → 2 → 3 → 4 → 5
E 降档: 5 → 4 → 3 → 2 → 1 → N → R
```

客户端上升沿检测，发送 `GearShiftC2SPacket`。

### 差速分配

```java
// TransmissionModel.PowertrainOutput
public record PowertrainOutput(double wheelRpm, double wheelTorque) {}
```

- 所有轮子**同转速**
- 扭矩**均摊**到每个轮子
- 悬挂方块的 P 控制器读取输出，转化为驱动力：

```java
double targetSpeed = targetRpm × 2π / 60 × wheelRadius;  // m/s
double speedError = targetSpeed - forwardSpeed;            // m/s
double baseGain = torqueGain × 0.2 × dt;                  // ← 增益从 0.02 改为 0.2 (10×)
double rawPCmd = speedError × baseGain × nm;

// ⚠ 扭矩上限：P 控制器力不能超过引擎能提供的极限
double maxWheelForce = torqueGain / max(rad, 0.01);
rawPCmd = clamp(rawPCmd, -maxWheelForce, maxWheelForce);
```
