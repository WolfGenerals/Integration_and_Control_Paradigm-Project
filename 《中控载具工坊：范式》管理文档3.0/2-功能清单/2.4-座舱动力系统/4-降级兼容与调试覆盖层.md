# 4. 降级兼容与调试覆盖层

### 降级兼容 (FallbackMode)

SubLevel 中**无驾驶舱**时，悬挂方块回退到固定值：

```java
FALLBACK_DRIVE_RPM = 400.0
FALLBACK_DRIVE_TORQUE = 80.0
```

### 调试覆盖层信息

上车时右下角显示（F3 风格，3 tick 刷新）：

```
重量: 12,340 kg             ← 服务端同步的实际物理质量
发动机: 3,200 RPM | 2,420 Nm  ← 当前转速 + 质量自适应扭矩（effectiveTorque）
档位/轮数: 3 档 | 4/4 轮着地
变速箱出: 2,000 RPM | 38,720 Nm
轮胎均值: 2,000 RPM | 9,680 Nm/轮
理想移速: 52.36 m/s
当前移速: 8.20 m/s (29.5 km/h)
摩擦预算: 84% 使用率         ← 腾空时自动显示 0%
```

> **注**：覆盖层显示的扭矩通过 `cockpit.getEffectiveTorque()` 获取，为 **质量自适应扭矩**。固定 30 Nm 仅作为初始回退值，每 tick 被 `effectiveTorque` 覆盖。
