# 2. 发动机模型

### 基础参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `ENGINE_IDLE_RPM` | 800 | 无油门时稳定怠速 |
| `ENGINE_MAX_RPM` | 6,000 | 红线转速 |
| `ENGINE_TORQUE` | 30 Nm | 基准扭矩（仅作为最小回退值和显示参考） |
| `TORQUE_WEIGHT_RATIO` | **0.02** | 扭矩/重量比：Nm 扭矩每 N 车重 |
| `TORQUE_IDLE_FRACTION` | **0.80** (06-08 从 0.35 调整) | 怠速时扭矩占峰值的比例。平坦化后极速时不再因耦合压低转速而扭矩崩溃 |
| `TORQUE_CURVE_SHARPNESS` | **1.0** | 扭矩曲线形状（越大峰值区越窄） |
| `RPM_ACCEL` | **200** RPM/tick | 无负载时加速率 |
| `RPM_DECEL` | **150** RPM/tick | 松油后自然减速率 |
| `RPM_LOAD_DECEL` | **80** RPM/tick | 负载拖拽减速率（乘 loadFactor） |
| `FINAL_DRIVE_RATIO` | **14.0** (06-08 新增) | 主减速比：所有档位齿比乘以该值得到最终传动比 |

### 质量自适应扭矩

发动机不再使用固定扭矩，而是根据车辆实际质量动态计算：

```java
double totalMass = massTracker.getMass();           // 车辆实际总质量 (kg)
double computedTorque = totalMass * 9.81 * TORQUE_WEIGHT_RATIO;
this.effectiveTorque = Math.max(computedTorque, ENGINE_TORQUE * 0.5);
```

**物理含义**：
- 轻车（~50 kg）：扭矩 ≈ 10 Nm（小扭矩，符合直觉）
- 标准测试车（70 kg）：扭矩 = 15 Nm（被 `ENGINE_TORQUE*0.5` 下限兜底）
- 重车（~12,000 kg）：扭矩 ≈ 2,354 Nm（大扭矩，力大砖飞）
- 保证**功率/重量比一致**：所有车获得相同的推重比体验

### 连续油门系统

用 `throttleLevel`（0.0~1.0 连续值）取代旧版布尔油门：

```java
if (direction > 0) this.throttleLevel = min(1.0, this.throttleLevel + THROTTLE_RATE);
else if (direction < 0) this.throttleLevel = max(0.0, this.throttleLevel - THROTTLE_BRAKE_RATE);
else this.throttleLevel = max(0.0, this.throttleLevel - THROTTLE_DECAY);

targetRpm = ENGINE_IDLE_RPM + (ENGINE_MAX_RPM - ENGINE_IDLE_RPM) * this.throttleLevel;
```

| 参数 | 值 | 含义 |
|------|-----|------|
| `THROTTLE_RATE` | 0.025/tick | 油门加速率（约 2 秒 0→1） |
| `THROTTLE_DECAY` | 0.015/tick | 松油后自动滑行衰减（约 3.3 秒 1→0） |
| `THROTTLE_BRAKE_RATE` | 0.04/tick | S 键主动减油（约 1.25 秒 1→0） |

### 扭矩平衡负载模型

转速变化率由扭矩余额决定：

```java
loadFactor = Σ(力需求) / Σ(引擎最大可输出力)

// 空档（currentGear == 0）：直接全速升降，不应用负载因子
// 档位中（currentGear != 0）：负载因子调节净加速度
double torqueRatio = 1.0 - clamp(loadFactor, 0, 5);
double netAccel = RPM_ACCEL * torqueRatio;  // 仅在档位中生效
```

**关键区别**：
- **空档（N）**：`netAccel = RPM_ACCEL`，油门直接驱动转速变化，不受负载影响
- **在档位中**：`netAccel = RPM_ACCEL × torqueRatio`，负载因子调节净加速度
  - 负载因子 < 0.3 → 引擎轻松 → 快速拉升
  - 负载因子 0.3~1.0 → 有负担 → 加速减缓
  - 负载因子 > 1.0 → 过载 → 转速被拖降

**06-08 修正**：loadFactor 分母除以轮数，不再被低估 4×。

### 发动机-轮速耦合（06-08 新增）

在档位中，发动机转速不能低于 `轮速×齿比×主减速比`：

```java
if (currentGear != 0) {
    double avgWheelRpm = getAverageWheelRpm(sl);
    double coupledRpm = avgWheelRpm × |gearRatio| × FINAL_DRIVE_RATIO;
    double minRpm = max(coupledRpm, ENGINE_IDLE_RPM);
    if (engineRpm < minRpm - 50) engineRpm = min(minRpm, ENGINE_MAX_RPM);
}
```

### 换挡 RPM 同步（06-08 新增）

```
engineRPM_new = engineRPM_old × newRatio / oldRatio
```

### 指令转速分离（06-08 新增）

有油门时使用油门指令转速而非实际发动机转速计算目标轮速。

### 扭矩曲线（06-08 新增）

```java
double rpmRatio = (engineRpm - ENGINE_IDLE_RPM) / (ENGINE_MAX_RPM - ENGINE_IDLE_RPM);
double shapedRatio = pow(rpmRatio, TORQUE_CURVE_SHARPNESS);  // ^sharpness 幂运算
double curveMultiplier = TORQUE_IDLE_FRACTION + (1.0 - TORQUE_IDLE_FRACTION) * sin(π × shapedRatio);
this.effectiveTorque *= curveMultiplier;
```

> 注：`TORQUE_CURVE_SHARPNESS=1.0` 时 `shapedRatio = rpmRatio`，公式简化为 `sin(π × rpmRatio)`。
> 增大 SHARPNESS 使峰值区更窄，低转扭矩更低、高转衰减更快。

| 参数 | 值 | 含义 |
|------|-----|------|
| `RPM_ACCEL` | 200 RPM/tick | 无负载时加速率 |
| `RPM_DECEL` | 150 RPM/tick | 松油自然减速率 |
| `RPM_LOAD_DECEL` | 80 RPM/tick | 负载拖拽减速率（乘 loadFactor） |
