# 文档管理指南

> 项目文档体系的**元规则** — 在修改任何文档前，请先阅读本目录下的文件。

## 目录

| 编号 | 文件 | 一句话说明 |
|:----:|------|-----------|
| 0.1 | [文档更新规则](0.1-文档更新规则.md) | 文档维护规范与单一事实来源原则 |

## 阅读策略

- **新协作者**：先读 `0.3-开发工作流规范.md` 了解协作红线，再读 `0.1-文档更新规则.md` 掌握文档维护规范
- **找文档**：使用 `0.2-文档结构总览.md` 的"需求场景"速查表
- **改进文档**：参考 `0.4-文档体系评估与改进建议.md` 了解当前文档体系的缺口和补全计划
