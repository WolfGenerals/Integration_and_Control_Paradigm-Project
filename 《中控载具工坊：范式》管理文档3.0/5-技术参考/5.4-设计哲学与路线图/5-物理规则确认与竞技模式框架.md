# 5. 物理规则确认与竞技模式框架

## 物理规则确认（2026-06-06）

### 规则 1：摩擦预算限制扭矩
P 控制器说「我要给这么多力」→ 摩擦圆说「你最多只能给这么多」→ 超出预算的部分 → 轮胎打滑。

### 规则 2：极速由齿比决定
因为没有空气阻力，载具加速到 P 控制器目标速度后进入巡航。

### 规则 3：当前没有「5 档地板油弹离合」的熄火机制
发动机转速是独立变量，不受车速约束。（远期通过引擎负载耦合修复。）

## 竞技模式框架

### 三种模式

```
默认模式（简单）        竞技模式（公平）        沙盒模式（创造）
──────────────────────────────────────────────────────────────
默认齿比               锁定齿比              可调齿比
预设驱动力             锁定驱动力            可调驱动力
方块上限 128           方块上限 128           无上限
悬挂预设               悬挂预设              全参数可调
轮胎默认               轮胎简化模型          未来：轮胎全量模型
物理全量               物理简化              物理全量
无需调参               规则约束              自由创造
```
