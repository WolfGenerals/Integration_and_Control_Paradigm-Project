# 4. 动力系统现状与进化路线

## 引擎-车轮脱耦问题

当前发动机转速与车轮转速之间**没有机械耦合**。问题：换挡时发动机转速不变，P 控制器反而按新的目标转速加速——**升档反而加速，降档反而减速**，与真实直觉完全相反。

## 换挡 RPM 同步（已确认方案）

```java
// gearUp() / gearDown() 末尾：
double oldRatio = getRatio(oldGear);
double newRatio = getRatio(this.currentGear);
this.engineRpm = Mth.clamp(this.engineRpm * (newRatio / oldRatio),
                             this.idleRpm, this.maxRpm);
```

if (clutchEngaged) {

    // 空档：自由空转（当前模式）
