# 2. 前 10 分钟用户旅程

```
0:00 ── 放下驾驶舱、四个轮子（简易合成配方或有创造模式标签页）
0:10 ── 右键驾驶舱 → 物理化 SubLevel → 自动配置默认参数
0:15 ── F 上车 → 显示 HUD（档位/转速/速度）
0:30 ── WASD 开走，Q/E 换挡
          ├─ 1 档推背感强但很快断油 → 换 2 档
          ├─ 2 档继续加速 → 换 3 档
          └─ 5 档达到 100 km/h 巡航
1:00 ── 玩出心得 → 按 C 打开调参 GUI
          ├─ 发现驱动力可调、齿比可改
          └─ 调到 60 → 「哇这车疯了一样」
3:00 ── 开始琢磨换挡时机、调参策略
10:00 ─ 开始想造自己的车 → 更大、更重、更强
          └─ 自然进入深度内容：悬挂调校、重心分配、轮胎选择
```

## 关键设计约束

- **10 秒内必须能动**，不需要看说明书
- **HUD 默认不开启**（不吓新手），可设置一个按键打开
- **简单模式是默认模式**，高级模式藏在 C 键菜单里
- **所有高级参数随时生效**，切换模式只是换预设，不是开关功能
