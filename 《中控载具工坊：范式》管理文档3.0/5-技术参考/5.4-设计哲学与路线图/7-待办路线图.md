# 待办路线图

> 本文为单一事实来源。所有待办功能状态以此为准。
> 已完成功能见 `2-功能清单/2.1-功能总表.md`。

## 战略方向
- **Blocky car 路线**：放弃等待 Sable 缩放，以 Minecraft 格子为基准，先出可玩成品。
- **资产策略**：不做官方的音效/特效/纹理包，做素材包导入框架，交给社区。

## 优先级说明
| 标记 | 含义 |
|:----:|------|
| ⭐⭐⭐ | 核心玩法必需 |
| ⭐⭐ | 重要体验提升 |
| ⭐ | 锦上添花 |
| ⚙️ | 调参/优化 |
