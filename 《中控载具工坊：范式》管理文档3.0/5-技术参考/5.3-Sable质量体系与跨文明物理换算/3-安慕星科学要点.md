# 3. 安慕星科学要点

## 重力异常

从玩家跳跃高度（约 1.25 方块）和滞空时间（约 0.6 s）反推：

$$ h = \frac{1}{2} g t^2 \Rightarrow g_{MX} \approx \frac{2 \times 1.25}{0.3^2} \approx 28 \text{ MXm/s}^2 $$

**安慕星重力大约是地球的 2~3 倍。**

## 缺失的阻尼项

当前载具物理缺少空气阻力。对载具行为的影响：

| 现象 | 安慕星解释 |
|------|-----------|
| 轻踩油门就 300 km/h | 安慕星真空环境下，加速没有空气掣肘 |
| 松油后减速慢 | 真空中滑行，自然损耗小 |
| 容易飞起来 | P 控制器全力输出时超过重力约束 |

## 时间离散性

安慕星的时间是离散的（20 tick/s）。物理引擎在每 tick 内计算一次力和运动：

- 理论上最高可控频率 = 20 Hz
- 比地球连续时间的控制系统更「粗糙」
- P 控制器的 `dt = 0.05` 就是 tick 间隔

### 数值稳定性限制

P 控制器的 `dt = 0.05` 不是随便选的——它是 tick 间隔。如果 P 增益过大，系统会发散：

```java
// 稳定条件：k_p × dt < 2 （对于显式欧拉积分）
// k_p = driveGain × 0.02
// → driveGain × 0.02 × 0.05 < 2
// → driveGain < 2000 ✅ 当前默认 30 远在安全范围内
```
