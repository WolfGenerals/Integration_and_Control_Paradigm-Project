# 4. 安慕星单位制 (MX Unit System)

## 基本单位

| 量 | 单位 | 符号 | 定义 |
|----|------|------|------|
| 长度 | MX-meter | MXm | 1 方块边长 |
| 体积 | MX-cube | MXm³ | 1 方块体积 |
| 质量 | MX-ton | MXT | 1 标准泥土方块质量 |
| 时间 | MX-second | MXs | ≈ 1 秒，离散为 20 tick |
| 速度 | MX m/s | MXm/s | m/s 视觉一致 |

## 导出单位

| 量 | 推荐单位 | 说明 |
|----|---------|------|
| 转速 | RPM | 沿用地球单位（已足够直观） |
| 车速 | km/h | 界面沿用 |
| 驱动力增益 | — | 无量纲，标为「MX 驱动力」 |
| 物质密度 | MXT/MXm³ | 即 Sable 系数，数值上 = 系数 |

## 代码与显示规范

| 场景 | 旧写法 | 新规范 |
|------|--------|--------|
| 变量名 | `torque` / `torqueGain` | `driveGain` |
| UI 标签 | "Nm" | "驱动力" |
| UI 质量 | "kg" | "MXT" 或 "吨"（可选） |
| 日志输出 | `... Nm` | `... (MX)` |
| 注释 | "扭矩" | "驱动力增益" |
