# 十二、自适应摄像机

### 设计目标
载具尺寸和姿态变化时，定值摄像机参数效果不佳。自适应系统根据 SubLevel 物理边框自动调整。

### 数据来源
```java
BoundingBox3dc bbox = clientSubLevel.boundingBox();
// bbox 返回世界空间 AABB（已包含 pose 变换）
double lenX = bbox.maxX() - bbox.minX();
double lenY = bbox.maxY() - bbox.minY();
double lenZ = bbox.maxZ() - bbox.minZ();
```

### 自适应高度
```java
if (CAMERA_ADAPTIVE_HEIGHT) {
    focusY = bbox.maxY() + 1.0 + CAMERA_HEIGHT_OFFSET;
    // 即使偏移为 0，也比车顶高 1 格
}
```

### 自适应距离
```java
if (CAMERA_ADAPTIVE_DISTANCE) {
    double longestSide = Math.max(lenX, Math.max(lenY, lenZ));
    distance += longestSide / 2.0;
    // 半边保证完整包裹
}
```

### 关键点
- `SubLevel.boundingBox()` 返回世界空间包围盒，直接读取无需额外变换
- 两个开关独立，互不依赖
- 自适应开启时手动偏移作为额外值叠加，不是替换
