# 十五、连续油门系统

### 设计目标
替换旧版布尔油门（throttleForward/Backward），实现平滑加减速。

### 工作机制

```java
// W 按下 → +0.025/tick（约 2 秒从 0→1）
// S 按下 → -0.04/tick（主动减油，约 1.25 秒从 1→0）
// 松开 → THROTTLE_DECAY=0.015/tick 自动滑行归零

int direction = scanThrottleDirection(sl); // +1=W按下, -1=S按下, 0=无操作
if (direction > 0)  throttleLevel = min(1.0, throttleLevel + THROTTLE_RATE);
else if (direction < 0)  throttleLevel = max(0.0, throttleLevel - THROTTLE_BRAKE_RATE);
else throttleLevel = max(0.0, throttleLevel - THROTTLE_DECAY);

// 目标转速 = 怠速 + (红线-怠速) × 油门位置
targetRpm = ENGINE_IDLE_RPM + (ENGINE_MAX_RPM - ENGINE_IDLE_RPM) * throttleLevel;
```

### 06-08 更新：三段式油门

| 参数 | 值 | 含义 |
|------|-----|------|
| `THROTTLE_RATE` | 0.025/tick | W 按下：加速（0→1 约 2 秒） |
| `THROTTLE_DECAY` | 0.015/tick | 松开：自动衰减滑行（1→0 约 3.3 秒） |
| `THROTTLE_BRAKE_RATE` | 0.04/tick | S 按下：主动减油（1→0 约 1.25 秒） |

三段式：W 按住 → 加速；无输入 → 滑行归零；S 按住 → 快速减油。

### 搜索方式
```java
// CockpitBE.tick() 直接扫描 SubLevel 内所有悬挂方块
// 读取 isThrottleForward() / isThrottleBackward() 布尔值
// 有 W 按下的方块 → direction = +1
// 有 S 按下的方块 → direction = -1
// 同时有 W 和 S → direction = 0（互斥）
```
