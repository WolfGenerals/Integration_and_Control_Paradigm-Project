# 四、摩擦圆物理

### 核心公式
```
frictionBudget = μ × max(springImpulse, nm × dt × MIN_IMPULSE_MULTIPLIER)
totalDemand = √(longForce² + latForce²)
if totalDemand > frictionBudget: 等比例缩减
```

### ⚠ 摩擦预算不能含阻尼
```java
double springImpulse = Math.abs((rest - slen) * ss * dt);  // ✅ 只用弹簧静载
// double normalImpulse = Math.abs(sf);                     // ❌ 含阻尼会塌陷
```

### ⚠ 单位必须匹配
- `Sable.HELPER.getVelocity()` 返回 **m/s**
- 物理 tick 中 targetSpeed 计算：`RPM × 2π / 60 × rad`（**m/s**）
- 视觉 tick 才需要 `/20.0` 转 m/tick

### ⚠ 缩放用 nm 而非 sm
- `sm = 20 × min(nm, 10)` — 上限 200，对 30 吨战车几乎为零
- 滚动阻力、侧滑阻尼、P 控制器增益都用 `nm`（真实质量块数）
