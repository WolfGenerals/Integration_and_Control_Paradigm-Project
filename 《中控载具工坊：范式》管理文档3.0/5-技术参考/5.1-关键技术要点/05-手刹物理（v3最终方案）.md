# 五、手刹物理（v3 最终方案）

手刹 = **轮子抱死** + **纯滑动摩擦**。

```java
double brakeMag = BRAKE_STRENGTH * mu * springImpulse; // ← springImpulse，不是 frictionBasis！
longForce = -(forwardSpeed / totalSpeed) * brakeMag;
latForce  = -(lateralSpeed / totalSpeed) * brakeMag;
```

- 刹车力基数 = `springImpulse`（真实法向冲量 ≈ nm × g × dt）
- 不能用 `frictionBasis`（被 `MIN_IMPULSE_MULTIPLIER=500` 膨胀 51 倍；06-08 已降为 30，差距缩小但原则不变）
- 减速度 ≈ 0.35g（`BRAKE_STRENGTH=0.5`），与车重无关
