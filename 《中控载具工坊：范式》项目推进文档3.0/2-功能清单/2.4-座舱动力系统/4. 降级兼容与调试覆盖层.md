# 4. 降级兼容与调试覆盖层

### 降级兼容 (FallbackMode)

SubLevel 中**无驾驶舱**时，悬挂方块回退到固定值：

```java
FALLBACK_DRIVE_RPM = 400.0
FALLBACK_DRIVE_TORQUE = 80.0
```
