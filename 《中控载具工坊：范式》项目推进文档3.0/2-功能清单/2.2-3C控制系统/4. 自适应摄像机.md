# 4. 自适应摄像机

### 设计目标
载具尺寸和姿态变化时，定值摄像机参数效果不佳。自适应系统根据 SubLevel 物理边框自动调整。

### 数据来源
```java
BoundingBox3dc bbox = clientSubLevel.boundingBox();
// bbox 返回世界空间 AABB（已包含 pose 变换）
double lenX = bbox.maxX() - bbox.minX();
double lenY = bbox.maxY() - bbox.minY();
double lenZ = bbox.maxZ() - bbox.minZ();
```

### 自适应高度
```
开启：focusY = bbox.maxY + 1.0 + CAMERA_HEIGHT_OFFSET
关闭：focusY = renderPos.y + CAMERA_HEIGHT_OFFSET（原版行为）
```
- `bbox.maxY`：SubLevel 物理边框最高点的世界 Y 坐标
- `+1.0`：始终比载具最高点高出 1 格
- 即使 `CAMERA_HEIGHT_OFFSET=0`，镜头也确保比车顶高 1 格

### 自适应距离
```
开启：distance = CAMERA_DISTANCE + longestSide / 2.0
关闭：distance = CAMERA_DISTANCE（原版行为）
```
- `longestSide = max(lenX, lenY, lenZ)`：边框最长边
- `longestSide / 2.0`：确保镜头能完整包裹整个载具

### 配置项
| 配置 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| `CAMERA_ADAPTIVE_HEIGHT` | true | — | 自适应摄像机高度开关 |
| `CAMERA_ADAPTIVE_DISTANCE` | true | — | 自适应摄像机距离开关 |
| `CAMERA_HEIGHT_OFFSET` | 0.0 | -5 ~ +5 | 焦点垂直偏移（自适应开启时为额外偏移） |
| `CAMERA_DISTANCE` | 4.0 | 0 ~ 60 | 摄像机距离（自适应开启时为额外距离） |
| `CAMERA_INVERT_Y` | false | — | 反转鼠标 Y 轴 |
