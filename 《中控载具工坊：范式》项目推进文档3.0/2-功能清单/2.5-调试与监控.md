# 调试与监控

## 车辆调试覆盖层 (DebugOverlay)

### 触发
上车后自动显示在右下角。

### 显示信息
```
重量: 12,340 kg
发动机: 3,200 RPM | 30 Nm
档位/轮数: 3 档 | 4/4 轮着地
变速箱出: 2,000 RPM | 48.0 Nm
轮胎均值: 2,000 RPM | 12.0 Nm/轮
理想移速: 52.36 m/s
当前移速: 8.20 m/s (29.5 km/h)
摩擦预算: 84% 使用率
```

### 实现
- 渲染：`RenderGuiEvent.Post`
- 刷新率：每 3 tick
- 关键原则：所有参数引用编译时常量，**不硬编码**数值

### 摩擦需求比显示（⚠ 06-08 更新）

```java
// 物理层：摩擦预算 = μ × max(弹簧静载, nm × dt × MIN_IMPULSE_MULTIPLIER)
// 06-08: MIN_IMPULSE_MULTIPLIER 从 500 降为 30，物理预算与显示预算的差距大幅缩小
// 显示层：自然预算 = μ × max(弹簧静载, nm × dt × 20)   ← ~2× 重量
this.frictionDemandRatio = totalDemand > 1e-10 ? totalDemand / Math.max(displayBudget, 1.0) : 0.0;
```

之前 `MIN_IMPULSE_MULTIPLIER=500` 时物理层摩擦预算被放大 ~50 倍，必须用自然预算做显示分母。现在 `MIN_IMPULSE_MULTIPLIER=30`，物理预算与显示预算仅差 1.5 倍，但为了直观反映"轮子空转"，仍沿用自然预算显示，值 > 100% 时表示力需求超过抓地极限。

## 网络包系统

基于 NeoForge 1.21.1 的 `CustomPacketPayload` + `StreamCodec` 体系。

### 包一览

| 包名 | 方向 | 频率 | 用途 |
|------|:----:|:----:|------|
| `SeatMountC2SPacket` | C→S | F 键按下 | 上车/下车请求 |
| `MountedStateS2CPacket` | S→C | 上下车时 | 通知客户端切换视角 + 同步 SubLevel UUID + vehicleMass |
| `VehicleControlC2SPacket` | C→S | 每 2 ticks | 实时控制输入（按键状态） |
| `VehicleKeyConfigC2SPacket` | C→S | 按键配置时 | 保存自定义按键绑定 |
| `GearShiftC2SPacket` | C→S | Q/E 按下 | 换挡请求 |
| `TurretTargetC2SPacket` | C→S | B 键 / 每 5 ticks | 炮塔瞄准目标坐标 (x,y,z) |
| `AnchorDataS2CPacket` | S→C | 锚点配置时 | 炮塔锚点 A 坐标 + 线条渲染数据 |
| `PlayerInputC2SPacket` | C→S | ~~每 20 ticks~~ | ❌ 已弃用（技术验证阶段产物） |

### 包定义示例

```java
// 使用 record + StreamCodec
public record MountedStateS2CPacket(boolean mounted, UUID subLevelUUID, double vehicleMass)
        implements CustomPacketPayload {

    public static final Type<MountedStateS2CPacket> TYPE = new Type<>(ID);
    public static final StreamCodec<RegistryFriendlyByteBuf, MountedStateS2CPacket> STREAM_CODEC =
            StreamCodec.composite(
                    ByteBufCodecs.BOOL, MountedStateS2CPacket::mounted,
                    UUIDUtil.STREAM_CODEC, MountedStateS2CPacket::subLevelUUID,
                    ByteBufCodecs.DOUBLE, MountedStateS2CPacket::vehicleMass,
                    MountedStateS2CPacket::new);
}
```

### 注册方式

```java
// 在 RegisterPayloadHandlersEvent 中注册
event.registrar("1.0")
    .playToServer(SeatMountC2SPacket.TYPE, STREAM_CODEC, SeatMountC2SPacket::handle)
    .playToClient(MountedStateS2CPacket.TYPE, STREAM_CODEC, MountedStateS2CPacket::handle);
```

### StreamCodec 限制
`StreamCodec.composite()` 最多支持 **8 个字段**。超过时需手动编解码：

```java
public static final StreamCodec<FriendlyByteBuf, MyPacket> CODEC = new StreamCodec<>() {
    @Override public MyPacket decode(FriendlyByteBuf buf) { /* 手动读取 */ }
    @Override public void encode(FriendlyByteBuf buf, MyPacket p) { /* 手动写入 */ }
};
```

### 发送方式

```java
PacketDistributor.sendToServer(payload);            // C→S
PacketDistributor.sendToPlayer(player, payload);     // S→C
```

### 键位

| 按键 | 功能 | 注册方式 |
|------|------|---------|
| F | 上车/下车 | ClientEvents 中监听 |
| C | 打开按键配置 GUI | `key.iac_p.vehicle_config` |
| Q | 升档 | ClientEvents 上升沿检测 |
| E | 降档 | ClientEvents 上升沿检测 |
| B | 炮塔射线检测/开火 | `key.iac_p.raycast_fire` + 持续瞄准每 5 tick |
