# 2. 摩擦圆物理模型 (FrictionCircle)

### 核心公式

每个 physics tick：

```
1. springImpulse = |(rest - slen) × ss × dt|     ← 弹簧静载冲量
2. minImpulse    = nm × dt × MIN_IMPULSE_MULTIPLIER
3. frictionBasis = max(springImpulse, minImpulse)
4. frictionBudget = μ × frictionBasis             ← 最大可用水平冲量
5. totalDemand = √(longForce² + latForce²)
6. if totalDemand > frictionBudget:
     scale = frictionBudget / totalDemand
     longForce ×= scale; latForce ×= scale        ← 摩擦圆约束
```

### ⚠ 关键教训：摩擦预算不能含阻尼

```java
// 错误：sf = (弹簧 + 阻尼) × dt
// 回弹时阻尼为负 → 抵消弹簧力 → frictionBasis 塌陷 → 驱动力归零
double normalImpulse = Math.abs(sf);          // ❌

// 正确：只用弹簧静载分量
double springImpulse = Math.abs((rest - slen) * ss * dt);  // ✅
```

### ⚠ 关键教训：单位必须匹配

```java
// 错误：Sable.HELPER.getVelocity() 返回 m/s
// targetSpeed 却用了 /20.0 转成 m/tick → 差 20 倍
double targetSpeed = activeRpm × 2π / 60 / 20 × rad;  // ❌

// 正确：去掉 /20.0
double targetSpeed = activeRpm × 2π / 60 × rad;        // m/s ✅
```

### 缩放规则

| 公式 | 随质量缩放？ | 上限 |
|------|:----------:|:----:|
| sm = 20 × min(nm, 10) | nm ≤ 10 线性增长 | 200 |
| P 控制器增益: `DRIVE_TORQUE × 0.02 × dt × nm` | ✅ 与 nm 成正比 | 无 |
| 滚动阻力: `ROLLING_RESISTANCE × nm × dt` | ✅ 与 nm 成正比 | 无 |
| 侧滑阻尼: `SIDE_SLIP_DAMPING × nm × dt` | ✅ 与 nm 成正比 | 无 |
| 摩擦预算: `mu × max(springImpulse, nm×dt×MULTIPLIER)` | ✅ 与 nm 成正比 | 无 |

> ⚠ 旧版用 `sm`（上限 200）→ 对 30 吨战车几乎为零。**必须用 `nm`**。

### 参数速查

| 常量 | 当前值 | 含义 | 调参方向 |
|------|--------|------|---------|
| `DRIVE_RPM` | 400 | 目标转速（降级模式） | 增大→极速提高 |
| `DRIVE_TORQUE` | 80 | P 控制器增益（降级模式） | 增大→加速更快 |
| `MIN_IMPULSE_MULTIPLIER` | **30** (06-08 从 500 调整) | 摩擦预算基数 | 增大→抓地力更强 |
| `TIRE_FRICTION_COEFFICIENT` | 0.9 | 轮胎摩擦系数 | 降低→更容易打滑 |
| `DEFAULT_CRR_BASE` | **0.035** (06-08 从 0.015 调整) | 滚动阻力基础系数 | 增大→减速更快 |
| `DRAG_COEFFICIENT` | **0.0045** (06-08 新增) | 二次方速度阻尼系数 | 增大→极速降低 |
| `CORNERING_STIFFNESS` | **20.0** (06-08 从 10.0 调整) | 侧偏刚度 | 增大→峰值更早、回正更强 |
| `SIDE_SLIP_DAMPING` | 6.0 | 侧滑抑制 | 增大→转向更稳 |
| `SPRING_STIFFNESS_PER_NM` | 400 | 弹簧刚度基数（N/m per nm） | 增大→所有车辆更硬 |
| `DAMPING_COEFF_PER_NM` | 10 | 阻尼系数基数（N·s/m per nm） | 增大→震荡衰减更快 |
| `SUSPENSION_MASS_THRESHOLD` | 5.0 | 质量自适应封顶阈值（nm） | 增大→更重的车才锁定刚度 |
