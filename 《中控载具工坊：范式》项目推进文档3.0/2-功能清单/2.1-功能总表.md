# 功能总表

所有功能的完整清单，按子系统分类，每个功能有独立命名和一句话说明。

> **⚠ 重要声明**：本表列出的是 **IAC-P 模组新增的功能**。所有功能均在前置模组提供的 API 之上构建：
> - **Sable** — Rapier 刚体物理引擎、SubLevel 系统、约束 API
> - **Simulated-Project** — SubLevel 装配/拆卸、chunk 扫描
> - **Offroad** — 车轮物品、轮胎数据组件 (`TireLike`)
> - **Create** — 应力网络、RPM 传输、KineticBlock 基类
> - **Aeronautics** — 航空物理模块
>
> 前置模组已有的功能（如 Offroad 的车轮物品和轮胎渲染、Create 的动力传输、Sable 的刚体碰撞）不在本表范围内。本模组的轮胎物理模型（胎压/刚度/滚动阻力/爆胎检测）为独立实现，在 Offroad 轮胎数据组件之上运行。
